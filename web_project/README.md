# 简单Web项目

这是一个使用Python HTTP服务器运行的简单静态网站项目。

## 项目结构

```
web_project/
├── index.html          # 主页面
├── about.html          # 关于页面
├── style.css           # 样式文件
├── script.js           # JavaScript文件
└── README.md           # 项目说明
```

## 功能特性

1. **响应式设计** - 适配各种屏幕尺寸
2. **现代UI** - 使用CSS Grid和Flexbox布局
3. **交互效果** - 包含JavaScript交互功能
4. **多页面** - 包含首页和关于页面
5. **Python服务器** - 使用内置HTTP服务器

## 快速开始

### 1. 启动服务器

在项目目录中运行以下命令：

```bash
# 使用Python 3
python -m http.server 8000

# 或者指定端口
python -m http.server 8080
```

### 2. 访问网站

在浏览器中打开：
- http://localhost:8000
- 或 http://localhost:8080（如果使用8080端口）

### 3. 停止服务器

在终端中按 `Ctrl + C` 停止服务器。

## 故障排除

### 服务器无法启动
- 检查Python是否正确安装
- 确保端口没有被占用
- 检查是否有防火墙阻止

### 页面无法访问
- 确认服务器正在运行
- 检查浏览器地址是否正确
- 查看终端是否有错误信息

### 样式不显示
- 检查CSS文件路径
- 查看浏览器开发者工具
- 确认CSS语法正确
