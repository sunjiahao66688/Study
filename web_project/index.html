这是一个使用Python HTTP服务器运行的静态网站示例

关于这个项目

这是一个简单的静态网站，展示了如何使用Python内置的HTTP服务器来托管网页。

Python的http.server模块提供了一个轻量级的HTTP服务器，非常适合开发和测试。

  • 响应式设计
  • 现代CSS布局
  • 卡片式UI设计
  • 渐变背景
  • 悬停效果

技术特性

这个网站使用了以下技术：

  • HTML5语义化标签
  • CSS3 Flexbox和Grid布局
  • CSS渐变和阴影
  • 响应式设计
  • Python HTTP服务器

所有代码都包含在一个HTML文件中，便于部署和维护。

如何使用

要运行这个网站，只需要：

python -m http.server 8000

然后在浏览器中访问：

http://localhost:8000

或者使用其他端口：

python -m http.server 8080
